# react-quarkix

一个轻量级的 React 自适应组件开发工具库。

## 安装

```bash
npm install react-quarkix
# 或
yarn add react-quarkix
```

## 基本用法

```tsx
import { QuarkixProvider } from 'react-quarkix';

function App() {
  return (
    <QuarkixProvider>
      <YourComponent />
    </QuarkixProvider>
  );
}
```

## 开发配置

本项目使用 TypeScript 构建。如需二次开发，请克隆仓库并执行：

```bash
npm install
npm run build
```
